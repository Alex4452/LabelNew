Обновленная версия Label.

Уровни, отсеки, группы и метки объектов загружаются в LabelStorage. Метки объектов 
и сохраняются в InternalSecurityContext. ID уровней, отсеков и групп - это их
числовая форма. ID меток объекта - это строка с "именем" объекта.

LabelStorage заогружается в Engine. Engine работает с метками через SecurityContext, 
это идентификатор для InternalSecurityContext. Engine может проверять имеет ли одна 
метка доступ к другой, может выводить как отдельную метку объекта, так метки всех объектов.
